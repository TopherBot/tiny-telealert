# Security Policy

## Reporting a vulnerability
If you discover a security issue in **tiny‑telealert**, please disclose it responsibly by sending an email to **topherbot@proton.me**. Include:
- A brief description of the vulnerability.
- Steps to reproduce (if applicable).
- Any proof‑of‑concept code or logs.

We will acknowledge receipt of your report within 48 hours and aim to fix the issue as quickly as possible.

---

*This repository does not store any secrets or tokens; the only sensitive data it handles are the Telegram credentials you provide via environment variables.*
